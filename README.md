# 羅茱輔助器

MapleStory Artale 羅密歐與茱麗葉 PQ 實驗室通道的即時協作輔助工具。

## 遊戲背景

實驗室通道共 10 層，每層有 4 扇門，每位玩家必須自己找出通往下一層的正確門。隊伍最多 4 人，各自走自己的路線，透過本工具即時共享進度。

## 功能

- **即時同步**：所有玩家標記的路徑即時顯示在同一畫面
- **順序解鎖**：必須完成當層才能進入下一層
- **排名系統**：率先走完 10 層者依序獲得名次
- **重置計數**：記錄已打了幾場，重置時自動清除 0 分玩家
- **自動清房**：超過 2 小時無活動的房間自動刪除

## 操作方式

| 操作 | 效果 |
|------|------|
| 左鍵 / 鍵盤 1–4 | 標記正確路徑（同步給所有人） |
| 右鍵 | 標記死路（僅本機顯示） |
| 點擊層號 | 切換當前操作層 |

## 使用方式

1. 開啟網頁，輸入**房間號碼**與**暱稱**
2. 將房間號碼分享給隊友
3. 各自進入後，嘗試傳點，左鍵標記正確的門

## 技術架構

- Vue.js 3（CDN）
- Tailwind CSS（CDN）
- Firebase Realtime Database（即時同步）
- GitHub Pages（靜態部署）
